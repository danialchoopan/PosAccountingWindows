# راهنمای نصب و راه‌اندازی

## پیش‌نیازها

- **ویندوز ۱۰ یا ۱۱**
- **.NET 9 SDK**

## نصب و اجرا

```bash
# نصب SDK
winget install Microsoft.DotNet.SDK.9

# اجرای برنامه
cd PosAccountingApp
dotnet run
```

## مرحله ۱: تنظیم اولیه (فقط اولین بار)

در اولین اجرا، پنجره تنظیم باز می‌شود:

| فیلد | توضیح |
|------|-------|
| نام فروشگاه | نام کسب‌وکار |
| نوع کسب‌وکار | سوپرمارکت / پوشاک / املاک / خودرو / مصالح |
| تلفن فروشگاه | شماره تماس |
| آدرس فروشگاه | آدرس |
| کد عبور مدیر | ۴ رقم (مثال: 1234) |
| درصد مالیات | پیش‌فرض ۱۰٪ |

روی **شروع به کار** کلیک کنید.

## مرحله ۲: ورود

کاربر **مدیر** ساخته شده است. کد عبوری که در مرحله ۱ وارد کردید را بزنید.

## مرحله ۳: اضافه کردن کاربران (اختیاری)

از سایدبار روی **مدیریت کاربران** کلیک کنید:
- روی **کاربر جدید** بزنید
- نام، کد عبور و نقش را انتخاب کنید
- روی **ذخیره** بزنید

## ریست کردن

```
%LOCALAPPDATA%\PosAccountingApp\pos_data.db
%LOCALAPPDATA%\PosAccountingApp\settings.json
```

این فایل‌ها را حذف کرده و برنامه را مجدداً اجرا کنید.
