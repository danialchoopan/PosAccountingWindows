# راهنمای نصب و راه‌اندازی

## پیش‌نیازها

- ویندوز ۱۰ یا ۱۱
- .NET 9 SDK

## نصب

```bash
winget install Microsoft.DotNet.SDK.9
cd PosAccountingApp
dotnet run
```

## اولین اجرا

### مرحله ۱: تنظیم اولیه
اطلاعات فروشگاه را وارد کنید و روی "شروع به کار" کلیک کنید.

### مرحله ۲: ورود
کد عبور مدیر را وارد کنید.

### مرحله ۳: شروع
وارد داشبورد شوید!

## اضافه کردن کاربر

از سایدبار روی "مدیریت کاربران" کلیک کنید و کاربر جدید بسازید.

## تنظیم تم

از صفحه تنظیمات، تم روشن/تاریک را تغییر دهید.

## خروجی و چاپ

در هر صفحه با جدول داده:
- دکمه PDF: خروجی PDF
- دکمه Excel: خروجی اکسل
- دکمه چاپ: چاپ مستقیم
- انتخاب تعداد ردیف: کشویی بالای جدول

## جزئیات ردیف

دوبار روی هر ردیف در جدول کلیک کنید تا جزئیات نمایش داده شود.

## ریست برنامه

```
%LOCALAPPDATA%\PosAccountingApp\pos_data.db
%LOCALAPPDATA%\PosAccountingApp\settings.json
```

این فایل‌ها را حذف کرده و برنامه را مجدداً اجرا کنید.
