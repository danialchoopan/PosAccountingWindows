# راهنمای نصب و راه‌اندازی سیستم مدیریت فروش و حسابداری

## پیش‌نیازها

- سیستم‌عامل **ویندوز ۱۰ یا ۱۱**
- **.NET 9 SDK** (نسخه ۹.۰ یا بالاتر)
- حداقل **۲ گیگابایت رم**
- فضای آزاد دیسک: **۵۰۰ مگابایت**

## مرحله ۱: نصب .NET 9 SDK

```bash
winget install Microsoft.DotNet.SDK.9
```

پس از نصب، کنسول را مجدداً باز کرده و تایپ کنید:
```bash
dotnet --version
```

## مرحله ۲: دریافت کدهای منبع

پوشه `PosAccountingApp` را در مسیر دلخواه کپی کنید.

## مرحله ۳: اجرا

```bash
cd PosAccountingApp
dotnet run
```

یا در Visual Studio روی **F5** کلیک کنید.

## مرحله ۴: راه‌اندازی اولیه

در اولین اجرای برنامه، **پنجره تنظیم اولیه** نمایش داده می‌شود:

| فیلد | توضیح |
|------|-------|
| نام فروشگاه | نام کسب‌وکار شما |
| نوع کسب‌وکار | سوپرمارکت / پوشاک / املاک / خودرو / مصالح |
| تلفن فروشگاه | شماره تماس |
| آدرس فروشگاه | آدرس محل کسب‌وکار |
| کد عبور مدیر | ۴ رقم (مثال: 1234) |
| درصد مالیات | پیش‌فرض ۱۰٪ |

پس از تکمیل روی **شروع کار** کلیک کنید.

## مرحله ۵: ورود به سیستم

پس از راه‌اندازی، پنجره **ورود** نمایش داده می‌شود:

| کاربر | کد عبور پیش‌فرض | نقش |
|-------|-----------------|------|
| مدیر سیستم | همان کدی که در مرحله ۴ وارد کردید | مدیر ارشد |
| کارمند ۱ | 1111 | صندوقدار |
| کارمند ۲ | 2222 | صندوقدار |
| حسابدار | 3333 | حسابدار |

نام کاربری را انتخاب کرده، کد عبور را وارد کنید و روی **ورود** کلیک کنید.

## مرحله ۶: استفاده از نرم‌افزار

پس از ورود، وارد **داشبورد** می‌شوید. از سایدبار سمت راست می‌توانید بخش‌های مختلف را باز کنید.

## ساخت نسخه اجرایی مستقل

```bash
dotnet publish -c Release -r win-x64 --self-contained -p:PublishSingleFile=true
```

فایل اجرایی در مسیر زیر ایجاد می‌شود:
```
bin\Release\net9.0-windows\win-x64\publish\
```

## عیب‌یابی

### خطا: "dotnet: command not found"
مسیر .NET SDK به متغیرهای محیطی اضافه نشده. کنسول را مجدداً باز کنید.

### خطا: "Build failed"
```bash
dotnet clean
dotnet restore
dotnet build
```

### ریست کردن کامل برنامه
فایل‌های زیر را حذف کنید:
```
%LOCALAPPDATA%\PosAccountingApp\pos_data.db
%LOCALAPPDATA%\PosAccountingApp\settings.json
```
سپس برنامه را مجدداً اجرا کنید تا مراحل راه‌اندازی اولیه دوباره نمایش داده شود.
