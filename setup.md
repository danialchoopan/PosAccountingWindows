# راهنمای نصب و راه‌اندازی

## پیش‌نیازها

- ویندوز ۱۰ یا ۱۱
- .NET 9 SDK

## نصب

```bash
winget install Microsoft.DotNet.SDK.9
cd PosAccountingApp
dotnet run
```

## اولین اجرا

### مرحله ۱: تنظیم اولیه
نام فروشگاه، نوع کسب‌وکار، تلفن، آدرس، کد عبور مدیر و درصد مالیات را وارد کنید.

### مرحله ۲: ورود
کد عبور مدیر را وارد کنید.

### مرحله ۳: شروع
وارد داشبورد شوید!

## اضافه کردن کاربر

از سایدبار روی "مدیریت کاربران" کلیک کنید و کاربر جدید بسازید.

## اضافه کردن دسته‌بندی

از سایدبار روی "دسته‌بندی کالاها" کلیک کنید. می‌توانید زیرمجموعه هم تعریف کنید.

## اضافه کردن کالا

از بخش "کالاها و خدمات" روی "کالای جدید" کلیک کنید و دسته‌بندی را از لیست انتخاب کنید.

## جستجو

Ctrl+G برای جستجوی سراسری در تمام بخش‌ها.

## خروجی و چاپ

در هر صفحه با جدول:
- دکمه PDF: خروجی PDF
- دکمه Excel: خروجی اکسل
- دکمه چاپ: چاپ مستقیم
- انتخاب تعداد ردیف: کشویی بالای جدول

## جزئیات ردیف

دوبار روی هر ردیف در جدول کلیک کنید.

## تنظیم تم

از صفحه تنظیمات: تم روشن/تاریک، اندازه فونت، کنتراست بالا.

## ریست برنامه

```
%LOCALAPPDATA%\PosAccountingApp\pos_data.db
%LOCALAPPDATA%\PosAccountingApp\settings.json
```

این فایل‌ها را حذف کرده و برنامه را مجدداً اجرا کنید.
