POS & Accounting Suite - Complete Feature List
=============================================

Date: 2026/06/16
Version: 1.0
Platform: .NET 9 / WPF / EF Core 9 / SQLite

COMPLETE FEATURE LIST
=====================

1. Initial Setup Wizard
   - Shop name, business type, phone, address
   - Admin PIN setup (4-digit SHA-256)
   - VAT percentage configuration

2. Multi-User Login System
   - User selection from dropdown
   - PIN code authentication
   - Role display (SuperAdmin, Admin, Cashier, Broker, Accountant)

3. User Management
   - Add/delete users with name, PIN, and role
   - Roles: SuperAdmin, Admin, Cashier, Broker, Accountant
   - Admin-only access for user management

4. Dashboard
   - Today's sales, revenue, customers, products
   - Pending cheques and low stock alerts
   - Colored stat cards (blue, green, orange, purple, red, teal)
   - Quick access buttons

5. POS (Point of Sale)
   - Product search by name or barcode in dropdown
   - Auto tax calculation (10%)
   - Payment methods in Farsi: Cash, Card, Ledger, Installments
   - Suspended invoice management
   - Change calculation

6. Product Management
   - Add/delete/edit products
   - Barcode, title, category, unit, purchase/sale price, stock
   - Farsi units: Number, Kilogram, Box, Meter, Ton, Bag, CubicMeter
   - Category dropdown from database

7. Category Management
   - Subcategories (parent/child)
   - 20 ready icons (Segoe MDL2 Assets)
   - Sort order

8. Customer Management
   - Ledger, credit limit, loyalty points
   - Add/delete customers

9. Cheque Treasury
   - Receivable/payable cheque tracking
   - Status: InVault, Passed, Returned

10. Expense Tracking
    - Category-based expense logging
    - Reports by date range

11. Reports
    - Daily/monthly sales
    - Profit/loss, inventory, customer reports

12. Settings
    - 5 color themes (Ocean Blue, Emerald Green, Royal Purple, Sunset Orange, Midnight Dark)
    - Font size adjustment (10-20)
    - High contrast accessibility mode
    - Shop profile, VAT, commission settings
    - Database backup/restore

13. Search
    - Global search (Ctrl+G)
    - Real-time filtering in all sections

14. Pagination
    - Selectable rows per page: 5/10/20/50/All
    - Hidden when table is empty

15. Export & Print
    - PDF export with Persian font
    - Excel export (.xlsx)
    - Direct printing (A4 landscape)

16. Detail View
    - Double-click row to see full details in popup

17. Barcode Scanner
    - USB barcode scanner support (HID keyboard mode)
    - Manual barcode entry
    - Scan history log

18. Accounting Engine
    - Chart of accounts (14 seeded accounts)
    - Journal entries with debit/credit
    - Balance checking

19. Supplier Debt Management
    - Supplier CRUD with contact info
    - Purchase recording (increases debt)
    - Payment processing (reduces debt)
    - Return processing
    - Ledger history per supplier

20. Customer Loyalty System
    - Bronze/Silver/Gold tier system
    - Points accumulation from purchases
    - Point redemption for discounts
    - Tier-based discount percentages
    - Transaction history

21. Color Themes (5 themes)
    - Ocean Blue: Professional blue accent
    - Emerald Green: Fresh green nature-inspired
    - Royal Purple: Elegant purple luxury
    - Sunset Orange: Warm orange energy
    - Midnight Dark: Modern dark with indigo

22. Accessibility
    - Font size adjustment (10-20)
    - High contrast mode
    - RTL layout with Vazirmatn font
