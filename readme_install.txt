مراحل نصب
=========

۱. فایل نصب را اجرا کنید
۲. مسیر نصب را انتخاب کنید (پیش‌فرض: Program Files)
۳. روی "نصب" کلیک کنید
۴. پس از نصب، برنامه را اجرا کنید

نیازمندی‌ها:
- ویندوز ۱۰ یا ۱۱ (۶۴ بیتی)
- حداقل ۲ گیگابایت رم
- ۵۰۰ مگابایت فضای آزاد

برای حذف برنامه:
- از منوی Start روی "حذف برنامه" کلیک کنید
- یا از Settings > Apps برنامه را حذف کنید
