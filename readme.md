# سیستم مدیریت فروش و حسابداری چندکسب‌وکاره

نرم‌افزار جامع مدیریت فروش، انبار، حسابداری و مشتریان برای کسب‌وکارهای ایرانی

---

## معرفی

نرم‌افزار حرفه‌ای و یکپارچه مدیریت فروشگاه و حسابداری برای محیط‌های کسب‌وکار ایرانی. پشتیبانی کامل از زبان فارسی، تقویم جلالی، محاسبه مالیات و گرد کردن ریالی.

## پشتیبانی از اصناف

| صنف | امکانات ویژه |
|------|-------------|
| سوپرمارکت | بارکد، وزن‌کشی، تاریخ انقضا، گرد کردن |
| پوشاک | رنگ، سایز، برند، فصل، موجودی ترکیبی |
| املاک | ثبت ملک، رهن/اجاره/فروش، محاسبه کارمزد |
| خودرو | ثبت خودرو، فروش نقدی/اقساطی/امانی |
| مصالح | واحد تناژی، حمل‌ونقل، دفتر پیمانکاران |

## ویژگی‌ها

### ورود و امنیت
- سیستم ورود چندکاربره با کد عبور
- نقش‌های مختلف (مدیر، صندوقدار، حسابدار، مشاور)
- راه‌اندازی اولیه با تنظیمات فروشگاه

### صندوق فروشگاهی
- اسکن بارکد و افزودن سریع کالا
- محاسبه خودکار مالیات و جمع فاکتور
- پرداخت نقدی، کارتخوان، نسیه، اقساطی
- نگهداری فاکتورهای معلق

### مدیریت کالاها
- ثبت بارکد، عنوان، دسته، قیمت
- ردیابی موجودی و هشدار کم بودن

### مدیریت مشتریان
- دفتر معین مشتری
- سقف اعتبار و امتیاز وفاداری

### خزانه‌داری
- ثبت چک‌های دریافتی و پرداختی
- پیگیری وضعیت چک

### هزینه‌ها
- دسته‌بندی هزینه‌ها
- گزارش‌گیری بر اساس بازه زمانی

### گزارشات
- فروش روزانه و ماهانه
- سود و زیان
- موجودی کالا

## راهنمای سریع شروع

```bash
# ۱. نصب .NET 9 SDK
winget install Microsoft.DotNet.SDK.9

# ۲. اجرای برنامه
cd PosAccountingApp
dotnet run

# ۳. در پنجره اول: اطلاعات فروشگاه را وارد کنید
# ۴. در پنجره دوم: کاربر را انتخاب کرده و وارد شوید
```

## کاربران پیش‌فرض

| کاربر | کد عبور | نقش |
|-------|---------|------|
| مدیر سیستم | (خودتان انتخاب می‌کنید) | مدیر ارشد |
| کارمند ۱ | 1111 | صندوقدار |
| کارمند ۲ | 2222 | صندوقدار |
| حسابدار | 3333 | حسابدار |

## کلیدهای میانبر

| کلید | عملکرد |
|------|--------|
| F1 | صندوق فروشگاهی |
| F2 | مدیریت کالاها |
| F3 | خزانه‌داری و چک |
| F4 | نگهداری فاکتور |
| F9 | گزارشات |

## ساختار فنی

- **پلتفرم:** .NET 9 / WPF
- **معماری:** MVVM
- **پایگاه داده:** SQLite + EF Core 9
- **رابط کاربری:** RTL فارسی، تم روشن
- **امنیت:** رمزنگاری SHA-256 کدهای عبور

## ریست کردن برنامه

فایل‌های زیر را حذف کرده و برنامه را مجدداً اجرا کنید:
```
%LOCALAPPDATA%\PosAccountingApp\pos_data.db
%LOCALAPPDATA%\PosAccountingApp\settings.json
```
